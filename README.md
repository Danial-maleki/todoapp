# Qt Todo Application

A simple, clean, and lightweight **Todo application** built with **C++ and Qt**.

This project is designed to demonstrate basic Qt concepts such as
UI design, signals & slots, and data handling in a desktop application.

---

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Simple and user-friendly UI
- Built with **Qt Widgets**
- Cross-platform (Linux / Windows / macOS)

---

## Requirements

- **Qt 5.15+** or **Qt 6.x**
- **C++17** or newer
- **Qt Creator** (recommended) or CMake toolchain

---

## Build & Run

### Using Qt Creator

1. Open the project `.pro` file in **Qt Creator**
2. Configure the build kit
3. Build and run the application

### Using CMake (if available)

```bash
mkdir build
cd build
cmake ..
make
